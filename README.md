## 💡 What It Does

Enter any password and the app instantly checks it against 5 security rules used in real-world systems. It gives you a score, a strength label, and specific tips on how to make it stronger.

---

## ✅ Security Checks

| Check | Description |
|---|---|
| **Length** | Must be at least 6 characters (10+ recommended) |
| **Uppercase** | At least one capital letter (A–Z) |
| **Lowercase** | At least one small letter (a–z) |
| **Numbers** | At least one digit (0–9) |
| **Symbols** | At least one special character (!@#$...) |

---

## 🎯 Features

- ✅ Real-time strength score (0–5)
- ✅ Color-coded result — Red (Weak) · Orange (Medium) · Green (Strong)
- ✅ Detects common leaked passwords like `123456` and `password`
- ✅ Personalized improvement suggestions
- ✅ Show / Hide password toggle
- ✅ Press Enter to check — no need to click

## 🚀 How to Run

```bash
git clone https://github.com/G1H6-bit/password-checker.git
cd password-checker
python password_checker.py
```
